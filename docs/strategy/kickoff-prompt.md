# Kickoff Prompt

We’re building `agency`, an AI growth OS for Shopify/D2C brands.

Start with the MVP architecture, product scope, schema design, and scaffold plan.

The product should connect Shopify, analytics, trends, content generation, approval flows, and social posting into one revenue-linked operating system.

Use this context:
- Positioning: revenue-linked content and growth OS for Shopify brands
- Initial wedge: weekly growth brief + hook generation + content planning tied to store performance
- Primary ICP: Shopify brands in roughly the `$1M-$20M` GMV range with lean teams in beauty, apparel, supplements, and home/lifestyle
- Key MVP modules: Analytics Agent, Content Agent, Trend Agent, Publishing Agent, Account Manager Agent
- MVP integrations: Shopify, Meta, GA4, and one social publishing workflow
- Build for approval-first automation before full autonomy
- Treat the current artist-booking repo only as architectural inspiration, not as the domain model
